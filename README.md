# 📝 ToDo App

A simple and stylish ToDo App built using **HTML**, **CSS**, and **JavaScript**. Add, view, and clear tasks in a minimal UI.

---

## 🚀 Features

- ➕ Add tasks
- 📋 Display all tasks
- 🔢 Task counter
- 🧹 Clear all at once
- 🎨 Clean, responsive design
